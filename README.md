# The Koide Relation as a Critical Fixed Point of Microscopic Spinorial Z3 Dynamics

This repository contains the LaTeX source, figures, and derivation scripts for a theoretical investigation of the Koide lepton mass relation.

The paper proposes that:

* the natural geometric variable is sqrt(m),
* arising from spinorial dynamics on a double-covered orbit,
* with an emergent Z3 structure,
* whose infrared critical point occurs at rho = sqrt(2),
* yielding the exact Koide relation Q = 2/3.

The work is exploratory and intended as a mathematical/field-theoretic proposal rather than an experimentally established model.

AI tools (Claude, ChatGPT, Grok) were used during editing, derivation checking, plotting, and manuscript preparation.
